# Multidynamic model for yield curve estimation

This directory contains the Jupyter notebooks and the LaTex source code for the study of multidynamic yield curve model. The directory is structured as follows:

- `data` directory contains the data used in the study. The data was sourced from Refinitiv Workspace using a license provided by Aalto University. The relevant terms and services apply.

- `figures` directory contains the figures used in the study

- `latex` directory contains the LaTex source code for building the study documentation

- `01_comp_of_term_structure_models.ipynb` Jupyter notebook contains the code used in the analysis in Chapter 5.1 in the study

- `02_info_in_the_MultidynamicModel_params.ipynb` Jupyter notebook contains the code used in the analysis in Chapter 5.2 in the study

- `Multidynamic_model_for_yield_curve_estimation.pdf` is the most recent draft for the study
