# Study on the dynamics governing yield curves

This directory contains the Jupyter notebooks and the LaTex source code for the study of yield curve dynamics. The directory is structured as follows:

- `data` directory contains the data used in the study. The data was sourced from Refinitiv Workspace using a license provided by Aalto University. The relevant terms and services apply.

- `figures` directory contains the figures used in the study

- `latex` directory contains the LaTex source code for building the study documentation

- `00_refinitiv_bond_history.ipynb` Jupyter notebook that can be used to source the data used in the study in Refinitiv's Codebook environment

- `02_expectations_hypothesis.ipynb` Jupyter notebook for testing the (pure) expectations hypothesis with a long time-series of observed yield curves
