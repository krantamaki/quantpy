# Study on the credit risk premiums

This directory contains the Jupyter notebooks and the LaTex source code for the study of credit risk premium. The directory is structured as follows:

- `data` directory contains the data used in the study. The data was sourced from Refinitiv Workspace using a license provided by Aalto University. The relevant terms and services apply.

- `figures` directory contains the figures used in the study

- `latex` directory contains the LaTex source code for building the study documentation

- `00_refinitiv_index_history.ipynb` Jupyter notebook that can be used to source the data used in the study in Refinitiv's Codebook environment

- `01_market_credit_spread.ipynb` Jupyter notebook for approximating the market credit spread, i.e., the non idiosyncratic credit premium
